🐳 Docker Image Commands
1️⃣ docker images

👉 Lists all images on your system.

Syntax:

docker images
2️⃣ docker pull

👉 Downloads image from registry (like Docker Hub).

Syntax:

docker pull <image>

Example:

docker pull nginx

3️⃣ docker build

👉 Creates image from Dockerfile.

Syntax:

docker build -t <name> .

Example:

docker build -t myapp .
4️⃣ docker rmi

👉 Removes an image.

Syntax:

docker rmi <image>

Example:

docker rmi nginx
